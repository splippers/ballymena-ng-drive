1. Project Overview

Goal

Create a historically accurate, driveable Morris Minor van (6cwt) as sold in Ballymena dealerships c. 1953-1972. Right-hand drive, period-correct trim.

3D Model

Use a base mesh or block out from blueprints. Focus on correct van body, split windscreen pre-1956, 8" drums, and NI registration plates.

Physics

JBeam with ~50 nodes, torsion bar front, leaf spring rear. A-Series engine spec with 4-speed gearbox. ~720kg curb weight.

4. Blender Commands & Mesh Setup

Blender Checklist

  1. 1 Import blueprints. Set Units: Metric, Scale 1.0. Van wheelbase = 2.184m
  2. 2 Model body. Separate objects: body, doors, hood, rear_doors, wheels, interior
  3. 3 Create empties for nodes: n0_L, n0_R, n1_L, n1_R etc. Snap to chassis hardpoints
  4. 4 UV unwrap. Materials named: morris_body, morris_glass, morris_interior
  5. 5 Apply Rotation & Scale. Origin to geometry center at Y=0 for BeamNG
  6. 6 Export: File > Export > Collada .dae. Check: Selection Only, Apply Modifiers
Blender Python Helper 
# Run in Blender Scripting tab
# Creates empties at common JBeam node locations for Morris Minor
import bpy

node_positions = {
    "n0_L": (-0.6, 1.7, 0.3),   # Front left chassis
    "n0_R": (0.6, 1.7, 0.3),    # Front right chassis
    "n1_L": (-0.6, -0.5, 0.3),  # Rear left chassis
    "n1_R": (0.6, -0.5, 0.3),   # Rear right chassis
    "n2_L": (-0.65, 1.6, 0.0),  # Front left suspension
    "n2_R": (0.65, 1.6, 0.0),   # Front right suspension
}

for name, pos in node_positions.items():
    empty = bpy.data.objects.new(name, None)
    empty.location = pos
    empty.empty_display_size = 0.1
    bpy.context.collection.objects.link(empty)

print("JBeam node empties created. Use these to snap mesh vertices.")

7. Testing & Debugging in BeamNG

In-Game Test Checklist

Common Errors

Vehicle falls through ground
Check node Z positions. Lowest node should be >0.2m. Add collision triangles in JBeam.
Spaghetti on spawn
Beam spring values too low. Increase to 2e6+ for chassis. Check for duplicate nodes.
Mesh not appearing
Verify .dae export. Check flexbody mesh name matches. Clear cache: /vehicles/ folder delete .cdae
Console spam "Invalid JSON"
Run through jsonlint.com. Common: trailing comma after last array item.
